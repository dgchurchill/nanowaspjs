About

NanoWasp is an emulator for the MicroBee computer. The MicroBee was popular in Australia in the mid 1980s. The model emulated is the 128k Disk System.

Acknowledgements

The Z80 CPU emulation code comes from Matt Westcott's JSSpeccy project, a ZX Spectrum emulator written in JavaScript. It was ported from the Fuse project.

Copyright

NanoWasp runs original MicroBee software. Although this software is now around 30 years old and no longer in use it is still covered by copyright. It is made accessible on this site only to preserve part of Australia's computing history.
